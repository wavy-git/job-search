# Senior Product Manager, Core Payments — Toast

Restaurant tech. Shape the systems and infrastructure managing payment flow across the Toast platform.

## Responsibilities
- Design/deliver payment capabilities (auth types, ML retry logic, dynamic routing)
- Define/prioritize roadmap for reliability/accuracy/efficiency
- Lead infrastructure programs (resiliency, migrations, failover)
- Interpret card scheme rules and processor specs
- Partner with engineering on transaction layer modernization (state mgmt, API contracts)
- Collaborate with finance/ops on settlement, reconciliation, cost modeling
- Manage processor partner integrations/certifications

## Required
- 5+ years PM in payments, fintech infrastructure, or transaction orchestration
- Proven track record shipping complex capabilities in core payment flows
- Strong technical fluency with payment specs and API definitions
- Card networks, processors, transaction lifecycle through settlement
- Distributed systems platform-level initiatives
